# Figma 업데이트 체크리스트

Figma에서 UI를 업데이트할 때 이 체크리스트를 따라하세요.

## ✅ 업데이트 전

- [ ] 현재 코드를 Git에 커밋
- [ ] 백업 브랜치 생성 (선택사항)
- [ ] `FIGMA_WORKFLOW.md` 문서 확인

## ✅ 업데이트 중

### 안전하게 교체 가능한 파일들:
- [ ] `components/ui/*` - UI 컴포넌트만 교체
- [ ] `components/figma/*` - Figma 컴포넌트만 교체
- [ ] `layouts/PortraitRetouchLayout.tsx` - 레이아웃만 교체

### 절대 건드리면 안 되는 파일들:
- [ ] `App.tsx` - 커스텀 로직 포함
- [ ] `features/*` - 모든 커스텀 기능
- [ ] `src/backend/*` - 백엔드 코드
- [ ] `package.json` - 프로젝트 의존성

## ✅ 업데이트 후

- [ ] `npm install` 실행 (새 의존성이 있는 경우)
- [ ] `npm run dev`로 테스트
- [ ] 모든 기능이 정상 작동하는지 확인
- [ ] Git에 변경사항 커밋

## 🔍 문제 발생 시

1. Git으로 이전 버전으로 되돌리기
2. `FIGMA_WORKFLOW.md`의 "주의사항" 섹션 확인
3. 레이아웃 props 인터페이스가 변경되지 않았는지 확인
